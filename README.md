# BVMT Intelligence

Assistant intelligent RAG + Agentic pour la Bourse des Valeurs Mobilières de Tunis.

## 🚀 Fonctionnalités
- Questions en langage naturel (français)
- Recherche hybride sur les rapports officiels BVMT
- Système multi-agents avec LangGraph
- Self-correction & réduction des hallucinations
- Interface Streamlit professionnelle
